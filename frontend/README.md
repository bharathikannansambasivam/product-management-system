# Product Management System

A responsive Product Management System built with **React**, **Tailwind CSS**, **Directus CMS**, and **PostgreSQL**. Instead of using the default Directus Admin Dashboard, this project provides a custom React interface for managing products.

## Features

- Responsive Home Page
- Product Listing
- Product Details
- Add Product
- Edit Product
- Delete Product
- Product Search
- Image Upload
- Form Validation (Formik & Yup)
- Duplicate Product Validation
- Toast Notifications
- Directus CMS Integration

## Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Axios
- Formik
- Yup
- React Hot Toast

### Backend

- Directus CMS
- PostgreSQL
- Directus File Storage
